# wheel of fortune

To launch - run `wheel.py` and interact with it from the command line.

__Command list:__

    c - Commands (list commands)
    i - Inventory (show)
    f - Guess puzzle
    p - Puzzle (show)
    r - Wheel (spin)
    v - Vowel (buy)
    # - Spy (mooshak)
    q - Quit (quit immediately)
